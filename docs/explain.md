
已知观察方向(C)为 `a*i + b*j + c*k`.
P 点为 `x*i + y*j + z*k`.

三点在同一条直线, 设定 `r`:

```
OQ = r * OC
```

`OQ` 和 `QP` 垂直, 那么点积为 `0`. 列方程求出 `r`

```c
float r = (a*x + b*y + c*z) / sumSquares3(a, b, c);
```

得到 `Q` 点. 也得到了 `QP` 向量.

`QP` 与 `QP` `OQ` 两者都垂直, 那么可以通过叉积算出来, 再重置其长度.

然后通过三角函数, 可以得到 `QP` 在 `Q-P-R` 平面上旋转任意角度的之后的向量.
